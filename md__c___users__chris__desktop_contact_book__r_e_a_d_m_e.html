MyContacts application

Address book where contacts can be added, edited and deleted.

Pre-installation

Ensure Java 8 and IntelliJ IDEA 2019.3 are installed on your system. IntelliJ IDEA is a recommendation but no must (Others: Eclipse 2019-03).

Running the programm

  1. Download the file from the repository

    Clone the repository on your local file system 

    ```
cd /path/to/install/location

git clone https://github.com/[your_username]/myContacts.git
```
  2. If you downloaded the zip-file, extract it.
  3. Open your IDE (IntelliJ IDEA or anything else).
  4. In the environment navigate to the project folder.
  5. Then navigate to the "Main" file and execute it.
